jQuery Syntax Highlighter

using this rather simple.

Setup your HTML


<pre><code data-lang='html'>
	<p>HTML Code to Highlight</p>
</code></pre>

Setup your Javascript


$(document).ready(function() {
	$('code').highlightSyntax();
});

Setup your CSS


table.syntax-highlighting tr td.code .const { color: #08c; }
table.syntax-highlighting tr td.code .operator { color: #fff; }
table.syntax-highlighting tr td.code .found { color: #ff0; }
table.syntax-highlighting tr td.code .comment { color: #f80; }
table.syntax-highlighting tr td.code .string { color: #0c0; }
table.syntax-highlighting tr td.code .regex { color: #80f; }
